# sample-doc — 가상 원고 위반 정답지

> ch-07-draft.md 는 copy-editor 적용 대상. 시작 시점에 심어둔 위반 목록(정답지).

## 의도적 위반 (수정 대상 ≥11건)

| 종류 | 건수 | 위치/예시 |
|------|:----:|-----------|
| 산문 끝 콜론 | 5 | "정해야 합니다:" / "다음과 같습니다:"(도입) / "아래와 같습니다:"(운영) / "요약하면 다음과 같습니다:"(마무리) / "결과:로" |
| Bold 한글 조사 위반 | 3 | `**마구(harness)**가` / `**컨테이너(container)**의` / `**검증(verification)**을` |
| 명백한 오탈자 | 3 | "했습다습다" / "의 의" / "그리고 그리고" |

## 예외 케이스 (수정 금지 ≥3건)

| 종류 | 위치 | 사유 |
|------|------|------|
| 시간 콜론 | "12:30" | 시간 표기 — 보존 |
| 코드 블록/인라인 콜론 | `model: sonnet` | 코드 표기 — 보존 |
| 표 헤더 콜론 | `| 항목: 값 |` | 표 헤더 — 보존 |

## 보류 후보 (모호)
- "결과:로 이어집니다" — 콜론 제거는 명백하나 뒤 조사 결합이 어색 → 처리 후 보류 메모 가능.
