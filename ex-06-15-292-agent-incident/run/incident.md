## 정량 보완 — N×(N-1)/2 sweep (책에 없는 보완)

만약 292명이 full mesh로 통신했다면:

| N | 잠재 채널 수 = N(N-1)/2 |
|---|------------------------|
| 50 | 1,225 |
| 100 | 4,950 |
| 200 | 19,900 |
| 292 | 42,486 |

각 채널이 평균 K개의 메시지를 운반한다면 메시지 총량은 42,486 × K. 큐 cap=50인 환경에서 한 팀원에 도달하는 평균 메시지 수는 (292-1) × K / 50 ≈ 5.82K — cap 즉시 포화.

메모리 추정: 36.8GB / 292 ≈ 126MB / agent. 책에서는 "RSS"라 명시 — 컨텍스트 윈도우·세션 메타·도구 호출 캐시 합산.
